# Comparativa: ChatGPT vs Claude

Análisis comparativo del uso de ChatGPT y Claude como asistentes de desarrollo.

## 1. Explicación de Conceptos Técnicos
Se solicitó la explicación de DOM, Event Loop y Closures.
* ChatGPT: Proporciona teoría estructurada y detallada.
* Claude: Ofrece respuestas más directas y ejemplos de código más aplicables a casos reales. Resulta más eficiente para consultas rápidas.

## 2. Detección de Errores
Se proporcionaron funciones con errores intencionados (omisión de JSON.parse y error de iteración).
* Ambos detectaron el error. Claude explicó con mayor precisión la causa del fallo y corrigió el código sin alterar la lógica original. ChatGPT tendió a reescribir la función completa.

## 3. Generación de Código
Se solicitó una función para filtrar y ordenar tareas pendientes.
* Ambos generaron código funcional. Claude utilizó métodos modernos de manipulación de arrays (filter y sort encadenados), mientras que ChatGPT ofreció una solución más conservadora.
