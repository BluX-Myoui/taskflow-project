# Registro de Prompts Efectivos

Recopilación de las diez estructuras de instrucciones más efectivas utilizadas durante el desarrollo:

1. Asignación de rol: "Actúa como un desarrollador Senior experto en JavaScript vanilla y revisa esta función."
2. Restricciones técnicas: "Crea una función para eliminar tareas. Restricción: Utiliza únicamente JavaScript puro, sin librerías externas."
3. Desarrollo iterativo: "Explica paso a paso cómo implementar la edición de tareas en el DOM, sin proporcionar el código final inmediatamente."
4. Formato Few-Shot: "Formatea las fechas. Ejemplo: entrada '2023-10-01' salida '1 de Octubre'. Aplica este formato a la siguiente variable."
5. Documentación: "Añade comentarios JSDoc a esta función detallando sus parámetros y valor de retorno."
6. Refactorización: "Refactoriza el siguiente bloque de app.js para mejorar su legibilidad y estandarizar la nomenclatura de variables."
7. Evaluación de casos límite: "¿Cuál es el comportamiento de esta función si el array de datos en LocalStorage es nulo?"
8. Accesibilidad: "Revisa la estructura HTML y sugiere atributos ARIA para cumplir con los estándares de accesibilidad."
9. Diseño de interfaz (UX/UI): "Actúa como experto en diseño de interfaces e indica un código hexadecimal que complemente el color principal para las acciones de eliminación."
10. Analogías simplificadas: "Explica el propósito de la etiqueta HTML template mediante una analogía sencilla."
